VLSI Processor February - June 2014

This was a project for the course "VLSI computer systems" on my faculty. Complete design and implementation of SoC in synthesizable VHDL. Processor contains processor core, instruction cache and data cache. Architecture is based on modern ARM RISC architecture. Core has four pipeline stages, instruction fetch, instruction decode, instruction execute and instruction writeback which allows cpu to execute four instructions together at any time. Core has load store architecture and all processor hazards are interlocked in hardware with maximum performances. SoC synthesized on Altera Cyclone II FPGA. For now source code is unavailable due to course rules.

Altera QuartusModelSimVHDL
